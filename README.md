# CODEC Project

CODEC est un logiciel permettant d'encoder / décoder facilement et rapidemment des fichiers par matrice G4C.

[Demo](https://maaximelh.github.io/codec-js/)

## Installation


```
git clone https://github.com/MaaximeLH/codec-js.git
```
